# HireLoop

AI-powered resume optimization platform built for modern ATS systems.

HireLoop analyzes your resume against job descriptions and generates:
- Tailored resumes
- ATS compatibility scores
- Personalized cover letters
- Skill gap analysis

---

## Features

• Resume upload and parsing  
• Job description analysis  
• AI-powered resume tailoring  
• ATS score generation  
• Cover letter generation  
• Resume export as PDF  

---

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS

Backend
- Node.js
- Express

AI
- Groq (LLaMA 3.1)

Other
- Puppeteer (PDF generation)

---

## How it Works

1. Upload your resume
2. Paste a job description
3. HireLoop analyzes ATS compatibility
4. AI generates an optimized resume and cover letter
